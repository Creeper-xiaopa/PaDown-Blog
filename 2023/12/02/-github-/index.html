Github 镜像加速

创建于 2023-12-02
GitHub 在国内经常会出现无法访问的情况，于是我自己搭建了几个镜像网站

1. Github 网站加速

该网站提供 Github.com 的 WebUI 镜像服务，不过无法登录
由于只是单纯的代理，部分 API 可能还是访问较慢

GhWeb.PaDown.eu.org

2. Github 文件加速网站

通过 Cloudflare Pages 构建
具体详见 “Github 文件加速 API”

GhProxy.PaDown.eu.org

3. Github 文件加速 API

使用 Cloudflare Worker 部署
Github 项目（可以自己部署）: /hunshcn/gh-proxy
暂不支持克隆

GhDown.PaDown.eu.org
使用 https://ghdown.padown.eu.org/ + Github项目地址 即可下载

示例（文件不存在）：
Branch 源码：github.com/Creeper-xiaopa/project/archive/master.zip
Release 源码：github.com/Creeper-xiaopa/project/archive/v1.tar.gz
Release 文件：github.com/Creeper-xiaopa/project/releases/download/v1/example.zip
Branch 文件：github.com/Creeper-xiaopa/project/blob/master/filename
